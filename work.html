PowerBI and Database Courses

The PowerBI was done as a final project for my Data Warehousing course. I had to add the appropriate columns from a given database, then find certain information that the professor was curious about, such as the top 5 and top 10 best and worst-selling products.

Also learned in the Data Warehousing course, was restoring a backed-up database in SQL Server, then connect that database with SSIS in Visual Studio. I was then able to successfully update the database using the SSIS data flowcharts I created.

In my Big Data Programming and Reporting course, I was taught how to use Hadoop with Apache Spark in Ubuntu. I also used pyspark in Jupyter Notebook with python.
